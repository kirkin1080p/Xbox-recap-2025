# 🧪 Chatterbox.quest – Tester Checklist (7 Day Cycle)

## How to use this

- Run this checklist **once per day for 7 days**
- Use the **same gamertag**
- Try to **play at least 1 game per day**
- If possible, vary behaviour (play 1 game, multiple games, or don’t play)

---

# ✅ Day Setup (Do This First)

- [ ] Connect your gamertag
- [ ] Confirm recap loads without errors
- [ ] Confirm profile image loads
- [ ] Confirm site does NOT crash or hang

---

# 🎮 Gameplay Tracking

## If you played today:

- [ ] Game you played appears in recap
- [ ] Last played game is correct
- [ ] No “apps” (YouTube, Netflix, etc.) counted as games
- [ ] Playing multiple games = all are tracked

## If you did NOT play:

- [ ] No fake game is shown
- [ ] System does NOT create a false play day

---

# 📅 Days Played System

- [ ] Days Played only increases when you actually played
- [ ] Reloading the site does NOT increase Days Played
- [ ] Going back the next day updates correctly

---

# 🔥 Streak System

- [ ] Streak increases if you play on consecutive days
- [ ] Streak does NOT increase if you skip a day
- [ ] Break days increase when you don’t play
- [ ] Longest streak updates correctly

---

# 🎯 Game Tracking

- [ ] Favourite game updates correctly over time
- [ ] Sessions increase only once per day per game
- [ ] Playing same game twice in a day does NOT double count
- [ ] New games are added correctly
- [ ] “One-hit wonder” games are tracked correctly

---

# 🧾 Journal System

- [ ] A new journal entry appears each day
- [ ] Entry matches what you actually played
- [ ] No-play days generate a “no activity” entry
- [ ] Entries do NOT duplicate
- [ ] Old entries remain unchanged

---

# 🏆 Achievements

- [ ] Achievement count loads (if account linked)
- [ ] Rarest achievement displays
- [ ] Recently unlocked achievements update (if applicable)

---

# 📊 Stats & Behaviour

- [ ] Most active day updates over time
- [ ] Monthly stats update correctly
- [ ] Peak day (most games played) updates correctly

---

# 🔄 Data Consistency

- [ ] Refreshing the page does NOT change stats randomly
- [ ] Stats remain the same across multiple visits
- [ ] No sudden resets of data
- [ ] No duplicated data

---

# 🔐 Account / Linking (Mandatory for tracking)

- [ ] Linking Xbox account works
- [ ] No errors during login
- [ ] Data improves after linking
- [ ] Signing out works correctly

---

# 💸 Support / Donations

- [ ] Donate stats load (if visible)
- [ ] No errors when loading support section

---

# ⚠️ Bugs / Issues

If anything breaks, log:

- What you did  
- What you expected  
- What actually happened  
- Screenshot (if possible)  

---

# 📅 End of 7 Days Check

After 7 days, confirm:

- [ ] Days Played matches actual days played
- [ ] Streak is correct
- [ ] Favourite game makes sense
- [ ] Journal tells a correct “story” of the week
- [ ] No missing days
- [ ] No fake days
- [ ] No duplicated entries

---

# TL;DR

Play games → come back daily → check stats match reality.

If anything feels wrong, it probably is.
